YS-Class Library
　

概要

YS-Class Libraryは、YSFLIGHT.COMからダウンロードできるプログラムのうちの大半のプログラムの基礎になっている、クラスライブラリで、YS Flight Simulatorでも使用している。主な機能は、次の通り。

  • 幾何計算
    • 2D/3Dベクトルクラス。基本的なオペレータも使用可能。
    • 3x3, 4x4, および一般的な行列クラス。
    • 2次元、3次元のポリゴンの内外判定。
    • 2D/3Dポリゴンの三角形分割。
    • ポリゴンシェル。集合演算も利用可能。
    • etc.
  • テンプレートクラス、テンプレート関数
    • Sorting
    • List
    • Array
    • etc.

このライブラリは、学部生時代から作って使ってきた関数、クラスを集めたもので、いくつかは、3次元グラフィックス開発キットBlue Impulseから移転して改良したものも含まれている。これまでに、多数の研究プロジェクトや、製品開発でこのライブラリを使用してきて、信頼性は高い。

著作権について

このクラスライブラリの著作権は、作者が保有するものとする。ただし、このライブラリを使用した結果について、使用者が全責任を負い、作者は責任を負わないという点に同意していただけるのであれば、フリー、商用、オープンソース、プロプライエタリを問わず、このクラスライブラリを利用していただいても構わない。この点に同意していただけない場合は、使用を認めない。また、このライブラリを利用したからといって、ライブラリを使用した製品のソースコードの公開を求めるなんてことは決してしない。

使用可能なプラットフォーム

ここでダウンロードできるパッケージは、ソースファイル、ヘッダファイル、および、Windows用ライブラリファイル (Visual C++ 2005 Express Editionでコンパイル)を含む。なお、このライブラリは、Linux (Cent OS5, Ubuntu 8.04で確認)、Mac OS Xでも使用可能で、その場合は、ターミナルで、srcサブディレクトリに移動して、次のようにタイプしてライブラリをBuildすることができる。

make -f makefile.unx

Buildに成功すると、ライブラリファイル libysclass.a ができる。
